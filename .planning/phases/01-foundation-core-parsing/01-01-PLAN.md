**src/types.ts** — All domain interfaces and type aliases:

```typescript
// Token usage from a single assistant message
interface TokenUsage {
  timestamp: Date;
  model: string;
  sessionId: string;
  inputTokens: number;        // Uncached input tokens
  outputTokens: number;       // Output tokens
  cacheCreationTokens: number; // Cache creation (billed at 1.25x or 2x)
  cacheReadTokens: number;    // Cache reads (billed at 0.1x, NOT rate-limited)
  cacheCreation5m: number;    // Ephemeral 5-minute cache creation tokens
  cacheCreation1h: number;    // Ephemeral 1-hour cache creation tokens
  cost: number;               // Calculated cost in USD
}

// Aggregated usage totals for a time bucket
interface AggregatedUsage {
  inputTokens: number;
  outputTokens: number;
  cacheCreationTokens: number;
  cacheReadTokens: number;
  totalCost: number;
  messageCount: number;
  firstMessage: Date | null;
  lastMessage: Date | null;
}

// Time bucket containers
interface TimeBuckets {
  session: Map<string, AggregatedUsage>;  // sessionId -> usage
  daily: Map<string, AggregatedUsage>;    // "YYYY-MM-DD" -> usage
  weekly: Map<string, AggregatedUsage>;   // "YYYY-Www" -> usage
  monthly: Map<string, AggregatedUsage>;  // "YYYY-MM" -> usage
}

// Plan types and their display names
type PlanType = 'pro' | 'max5' | 'max20';

interface PlanConfig {
  type: PlanType;
  displayName: string;
  monthlyPrice: number;
}

// Per-model pricing (configurable via settings)
interface ModelPricing {
  inputPerMillion: number;
  outputPerMillion: number;
  cache5mWriteMultiplier: number;  // 1.25
  cache1hWriteMultiplier: number;  // 2.0
  cacheReadMultiplier: number;     // 0.1
}

// Extension state persisted to globalState
interface PersistedState {
  version: number;  // Schema version for future migrations
  lastParseTimestamp: string | null;
  totalFilesProcessed: number;
  totalLinesSkipped: number;
  timeBuckets: SerializedTimeBuckets;  // Maps serialized as arrays
}

// Serializable version of TimeBuckets (Maps -> arrays of [key, value])
interface SerializedTimeBuckets {
  session: [string, AggregatedUsage][];
  daily: [string, AggregatedUsage][];
  weekly: [string, AggregatedUsage][];
  monthly: [string, AggregatedUsage][];
}

// Parse result from a single file
interface FileParseResult {
  filePath: string;
  records: TokenUsage[];
  linesSkipped: number;
  errors: string[];
}
```

Export all interfaces and types as named exports.

**src/parser/schemas.ts** — Zod schemas matching actual Claude Code JSONL format:

Based on actual JSONL structure observed:
- Top-level: { type, timestamp, sessionId, uuid, parentUuid, message?, ... }
- type "assistant" has message.usage with token counts
- message.usage.cache_creation has ephemeral_5m and ephemeral_1h breakdowns
- Fields like isSidechain, userType, cwd, version, gitBranch, slug exist but are not needed

Create schemas:
- UsageSchema: validates usage object with input_tokens, output_tokens, cache_creation_input_tokens (optional, default 0), cache_read_input_tokens (optional, default 0), cache_creation object (optional, with ephemeral_5m_input_tokens and ephemeral_1h_input_tokens)
- AssistantMessageSchema: validates full JSONL line for type "assistant" with message.model and message.usage
- Use z.passthrough() on objects to allow unknown fields without failing
- Export a parseAssistantMessage(line: string) function that returns TokenUsage | null

**src/utils/paths.ts** — Cross-platform path utilities:
- getClaudeProjectsDir(): returns path.join(os.homedir(), '.claude', 'projects')
- findAllSessionFiles(projectsDir: string): recursively finds *.jsonl in project dirs AND subagent dirs
  - Pattern: {projectsDir}/{projectName}/*.jsonl (top-level sessions)
  - Pattern: {projectsDir}/{projectName}/{sessionId}/subagents/*.jsonl (subagent files)
  - Use fs.promises.readdir with withFileTypes for efficiency
  - Handle errors gracefully (skip unreadable dirs, log warning)

**src/utils/logger.ts** — OutputChannel wrapper:
- Logger class wrapping vscode.OutputChannel
- Methods: info(), warn(), error() — all prefix with timestamp
- Static create(name: string) factory method
- Lazy channel creation (only create OutputChannel when first log happens)
- Export singleton pattern or factory
