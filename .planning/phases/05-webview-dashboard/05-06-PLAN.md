  <action>
Add the data transformation layer to DashboardProvider. This converts the extension's internal data structures into the webview-safe DashboardData format.

**Add imports:** Import `TimeBuckets`, `StatusBarData`, `AggregatedUsage` from `../types.js`. Import `DashboardData`, `RateLimitData`, `TrendDataPoint` from `./app/types.js`.

**Add instance fields:**
```typescript
private _buckets?: TimeBuckets;
private _statusBarData?: StatusBarData;
private _planType: string = 'pro';
private _activePeriod: 'daily' | 'weekly' | 'monthly' = 'daily';
```

**Add static method `buildDashboardData`:**
```typescript
public static buildDashboardData(
  buckets: TimeBuckets,
  statusBarData: StatusBarData,
  planType: string,
  activePeriod: 'daily' | 'weekly' | 'monthly' = 'daily'
): DashboardData
```

Implementation details for each DashboardData field:

1. **Token breakdown:** Use `statusBarData.totalInputTokens`, `statusBarData.totalOutputTokens`. For cache tokens, sum from today's daily bucket: get today's key (`format(new Date(), 'yyyy-MM-dd')`), look up in `buckets.daily`, extract `cacheCreationTokens` and `cacheReadTokens`. If no today bucket, use 0.

2. **Cost data:** Direct from `statusBarData`: `todayCost`, `monthCost`, `totalCost`.

3. **Rate limits:** Convert each `RateLimitInfo` from `statusBarData.rateLimits` to `RateLimitData`:
   ```typescript
   const convertRateLimit = (info: RateLimitInfo): RateLimitData => ({
     name: info.name,
     currentTokens: info.currentTokens,
     estimatedLimit: info.estimatedLimit,
     percentage: info.percentage,
     resetTime: info.resetTime?.toISOString() ?? null,
     isHit: info.isHit,
   });
   ```
   Apply to `session5h`, `weekly`, `weeklySonnet`.

4. **Session timing:** Compute from `statusBarData.rateLimits.session5h.resetTime`:
   - If `resetTime` exists: `windowExpiry = resetTime.toISOString()`, `windowStart = new Date(resetTime.getTime() - 5 * 60 * 60 * 1000).toISOString()`, `timeRemainingMinutes = Math.max(0, Math.round((resetTime.getTime() - Date.now()) / 60000))`
   - If null: all three fields are null

5. **Burn rate:** `tokensPerMinute = statusBarData.burnRate`. For `minutesUntilLimit`: if burnRate > 0 and session5h limit exists, compute `(estimatedLimit - currentTokens) / burnRate`. If burnRate = 0, set to null.

6. **Trend data:** Based on `activePeriod`, pick the corresponding Map from `buckets`:
   - `'daily'` -> `buckets.daily`
   - `'weekly'` -> `buckets.weekly`
   - `'monthly'` -> `buckets.monthly`
   Convert to sorted `TrendDataPoint[]`:
   ```typescript
   const bucketMap = buckets[activePeriod];
   const trendData: TrendDataPoint[] = Array.from(bucketMap.entries())
     .sort(([a], [b]) => a.localeCompare(b))
     .map(([period, agg]) => ({
       period,
       inputTokens: agg.inputTokens,
       outputTokens: agg.outputTokens,
       cacheCreationTokens: agg.cacheCreationTokens,
       cacheReadTokens: agg.cacheReadTokens,
       totalCost: agg.totalCost,
     }));
   ```

7. **Session comparison -- CRITICAL for SP-06:**
   - `currentSessionTokens`: Sum output tokens from all session buckets whose `lastMessage` is within the last 5 hours. Iterate `buckets.session.values()`, filter by `lastMessage >= fiveHoursAgo`, sum their `outputTokens`.
   - `averageSessionTokens`: Compute the mean output tokens across ALL sessions in `buckets.session`. Formula: `totalOutputAcrossAllSessions / buckets.session.size`. Iterate all entries in `buckets.session`, sum `outputTokens`, divide by `buckets.session.size`. If no sessions, use 0.
   - `sessionCount`: `buckets.session.size`

8. **Metadata:** `lastUpdated: statusBarData.lastUpdated.toISOString()`, `filesProcessed: statusBarData.filesProcessed`, `linesSkipped: statusBarData.linesSkipped`, `planType`.

**Add `updateBuckets` method:**
```typescript
public updateBuckets(buckets: TimeBuckets, statusBarData: StatusBarData, planType: string): void {
  this._buckets = buckets;
  this._statusBarData = statusBarData;
  this._planType = planType;
  const data = DashboardProvider.buildDashboardData(buckets, statusBarData, planType, this._activePeriod);
  this.updateData(data);
}
```

**Update `resolveWebviewView` message handler for `changePeriod`:**
When `changePeriod` message received:

   ```typescript
     this._buckets, this._statusBarData, this._planType, this._activePeriod
   );
   this.updateData(data);
   ```
