    Remove:

    ```typescript
    // TODO: Filter by model name once per-model weekly aggregation exists in TimeBuckets
    const weeklySonnetTokens = weekData?.outputTokens ?? 0;
    ```

    Replace with per-model filtering logic:
    ```typescript
    // Weekly Sonnet limit: Sum output tokens only from claude-sonnet-* models this week
    let weeklySonnetTokens = 0;
    for (const [key, agg] of buckets.modelWeekly.entries()) {
      // Key format: "YYYY-WII:model-name"
      if (key.startsWith(weekKey + ':') && key.includes('claude-sonnet')) {
        weeklySonnetTokens += agg.outputTokens;
      }
    }
    ```

    This iterates the modelWeekly map, matches entries for the current ISO week that
    contain "claude-sonnet" in the model name. This catches all Sonnet variants:
    - claude-sonnet-4-20250514
    - claude-3-5-sonnet-20241022
    - Any future sonnet model names

    The check uses `key.includes('claude-sonnet')` rather than exact prefix matching
    to handle both "claude-sonnet-*" and "claude-3-5-sonnet-*" naming patterns.

    **2. Verify the existing weekly calculation is unchanged:**
